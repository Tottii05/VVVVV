## Controles del Juego

| Tecla          | Acción                    |
|----------------|---------------------------|
| ⬅️ o **A**     | Mover a la izquierda      |
| ➡️ o **D**     | Mover a la derecha        |
| **Espacio**    | Cambiar gravedad          |
| **P**          | Abrir menú de pausa       |
